# 页面解析和数据提取
- 结构数据：先有的结构·再谈数据
    - JSON文件
        - JSON Path
        - 转换成Python类型进行操作(json类)
    - XML文件
        - 转换成python类型(xmltodict)
        - XPath
        - CSS选择器
        - 正则
- 非结构化数据：现有数据·再谈结构
    - 文本
    - 电话号码
    - 邮箱地址
        - 通常处理此类数据，使用正则表达式
    - Html文件
        - 正则
        - XPath
        - CSS选择器
